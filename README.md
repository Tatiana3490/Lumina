# Lumina: Echoes of the Soul

Prototipo del videojuego desarrollado con Phaser 3 y Vite.

## Requisitos
- Node.js instalado.

## Instalación
1. Abrir una terminal en esta carpeta.
2. Ejecutar:
   ```bash
   npm install
   ```

## Ejecución
Para iniciar el servidor de desarrollo y jugar:
```bash
npm run dev
```
Abre el navegador en la dirección que aparece (usualmente `http://localhost:5173`).

## Controles
- **Flechas / WASD:** Moverse.
- **Espacio / Flecha Arriba:** Volar (Aletear).
- **Mantener Espacio / Arriba:** Planear (Caída lenta).
- **Luz:** La luciérnaga ilumina automáticamente el entorno oscuro.
